# E-commerce проект для банковских операций

## Описание
Проект реализует базовую структуру интернет-магазина с классами `Product` и `Category` с использованием инкапсуляции, геттеров, сеттеров и класс-методов.

---

## Реализовано

### Версия 14.1
- Класс `Product` с атрибутами: `name`, `description`, `price`, `quantity`
- Класс `Category` с атрибутами: `name`, `description`, `products`
- Автоматический подсчёт количества категорий и товаров (атрибуты класса)
- Тесты с покрытием 100%

### Версия 14.2 (текущая)
- Приватный атрибут `__products` в классе `Category` (инкапсуляция)
- Метод `add_product()` для добавления товара в категорию с автоматическим увеличением счётчика товаров
- Геттер `@property` для атрибута `products`, возвращающий строку с товарами в формате:  
  `"Название продукта, X руб. Остаток: X шт.\n"`
- Приватный атрибут `__price` в классе `Product`
- Геттер и сеттер для цены с проверкой на положительное значение
- **Дополнительно**: при попытке понизить цену запрашивается подтверждение пользователя (`y/n`)
- Класс-метод `new_product()`, создающий продукт из словаря
- **Дополнительно**: в `new_product()` реализована проверка дубликатов по имени — при совпадении количество складывается, выбирается максимальная цена
- Тесты покрывают новую функциональность, общее покрытие — **92%**

### Пример использования

python

from src.product import Product
from src.category import Category

# Создание продуктов
product1 = Product("Смартфон", "Хороший телефон", 50000.0, 10)
product2 = Product("Наушники", "Беспроводные", 3000.0, 25)

# Создание категории
category = Category("Электроника", "Гаджеты", [product1, product2])

# Добавление нового продукта
product3 = Product("Чехол", "Силиконовый", 1000.0, 50)
category.add_product(product3)

# Просмотр товаров в категории (геттер)
print(category.products)

# Изменение цены с проверкой
product1.price = 45000.0  # понижение — запросит подтверждение

# Создание продукта через класс-метод из словаря
new_product = Product.new_product({
    "name": "Планшет",
    "description": "10 дюймов",
    "price": 25000.0,
    "quantity": 8
})
