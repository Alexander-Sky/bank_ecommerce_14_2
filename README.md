# E-commerce проект для банковских операций

## Описание
Проект реализует базовую структуру интернет-магазина с классами `Product` и `Category`.

## Реализовано в версии 14.1:
- Класс `Product` с атрибутами: name, description, price, quantity
- Класс `Category` с атрибутами: name, description, products
- Автоматический подсчёт количества категорий и товаров (атрибуты класса)
- Тесты с покрытием 100%

## Установка и запуск тестов
```bash
poetry install
poetry run pytest
